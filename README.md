# MyLibrary

MyLibrary is a simple and clean web-based library application that allows users to keep track of books they have read or plan to read. It provides an easy interface to add new books, remove books, and toggle their reading status.

## How It Works

MyLibrary uses a simple JavaScript Book constructor or class (depending on your implementation) to create book objects.  
Books are stored inside an array representing the current library.  
Each book can be displayed, toggled, or removed through DOM interaction.

## Usage

1. Open `index.html` in your browser.  
2. Add a book using the form.  
3. View books displayed in the library grid.  
4. Toggle read/unread status.  
5. Remove a book if needed.
